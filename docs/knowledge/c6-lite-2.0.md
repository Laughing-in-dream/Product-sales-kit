# C6 Lite 2.0（产品线 id: `c6lite20`）

> 预置套装流程：先在同一行的四个核心套装中选择“镜头数 + 通讯版本”，再确认强制包含及可选的电源线；单镜头套装可继续增加一只摄像头，最后选择 R-Watch 和 Micro SD 卡。

## 1. 向导流程

1. **选择核心套装**：四个入口卡片必须明确展示镜头数与通讯版本：
   - RS232 双镜头：`5154022100067`
   - RS232 单镜头：`5154022100068`
   - CAN 双镜头：`5154022100044`
   - CAN 单镜头：`5154022100083`
   同镜头数的 RS232 / CAN 主机外观相同，复用同一张主机图；桌面端四张卡片在同一行展示。
2. **选择电源线**：进入套装后直接确认电源线，使用 AD Plus 2.0 的“必选锁定 + 可选勾选”结构。
   - RS232：强制包含并锁定电源散线 `1261090100094`；16PIN OBD `1261090100095` 与 9PIN OBD `1261090100117` 可额外选择。
   - CAN：强制包含并锁定 16PIN OBD `1261090100120`；9PIN OBD `1261090100121` 与 CAN 电源散线 `1261090100122` 可额外选择。
3. **选择摄像头**：仅单镜头套装可增加 1 路 AHD 摄像头；选中后自动带 1 条 AHD 视频拓展线，并可选摄像头延长线。双镜头套装在此步骤仅提示已占满两路通道。
4. **选择附件**：选择 R-Watch 与最多 2 张 Micro SD 卡；每张 SD 卡独立选择容量 / 料号，结构复用 AD Plus 2.0。
5. **确认并导出**。
