# C6 Lite 2.0（产品线 id: `c6lite20`）

> 预置套装流程：先在四个明确的核心套装中选择“镜头数 + 通讯版本”，再选择已预设的电源线；单镜头套装可继续增加一只摄像头，最后选择 R-Watch 和 Micro SD 卡。

## 1. 向导流程

1. **选择核心套装**：四个入口卡片必须明确展示镜头数与通讯版本：
   - RS232 双镜头：`5154022100067`
   - RS232 单镜头：`5154022100068`
   - CAN 双镜头：`5154022100044`
   - CAN 单镜头：`5154022100083`
   同镜头数的 RS232 / CAN 主机外观相同，复用同一张主机图。
2. **选择电源线**：进入套装后直接选择电源线，使用 AD Plus 2.0 风格的卡片展示。
   - RS232 默认选中电源散线 `1261090100094`；仍可按实际车型改选 RS232 的 16PIN / 9PIN OBD 电源线。
   - CAN 默认选中 16PIN OBD 电源线 `1261090100120`；可按实际车型改选 CAN 的 9PIN OBD 电源线 `1261090100121`。
3. **选择摄像头**：仅单镜头套装可增加 1 路 AHD 摄像头；选中后自动带 1 条 AHD 视频拓展线，并可选摄像头延长线。双镜头套装在此步骤仅提示已占满两路通道。
4. **选择附件**：选择 R-Watch 与最多 1 张 Micro SD 卡；SD 卡的容量 / 料号选择使用 AD Plus 2.0 的统一结构。
5. **确认并导出**。
