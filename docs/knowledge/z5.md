## 1. 方案构成

向导流程（`js/04-product-meta.js: Z5_STEPS / Z5_STEP_ROWS`）：

1. 固定核心套装：Z5 Kit（row 2 / `5210056100004`），以与其他产品一致的核心套装卡片呈现。
2. 存储卡：64GB、128GB、256GB、512GB 四选一（rows 3–6）；使用统一的“数量 + 容量 / SKU”存储卡模板，数量上限为 1。
3. 系统连接图：Excel 的方案资源 `S1`（`6--Z5 System Connection Diagram.pdf`）应在核心套装页提供查看入口；它是安装参考，不加入 BOM。
4. 可选附件：Z5/ABS 电源盒异位安装时选择 1.3M 延长线（row 8 / `1262010100136`）；多门磁场景选择有线门磁（row 9 / `5163020100001`）。
5. 确认并导出。
