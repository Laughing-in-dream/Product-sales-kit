# 960C53（产品线 id: `960c53`）

> BSD 侧向盲区单品方案（C53 摄像头 + B2/B3 报警器），可独立工作或作为 AD Plus 2.0／M1N 2.0／M3N 的级联从机。

## 0. 模式与级联规则

- 向导首步沿用 AVM 的入口结构：先以两张产品组合图卡片选择 **Standalone** 或 **Cascade**，再在同页的精简套装区选择**一套** C53-L 或 C53-R。首页仅展示 4 个**无 Logo**版本；带 Streamax Logo 的两个版本不在首页展示。B3、短支架、电源盒、视频输出线、串口输入线和螺丝刀均为套装内含物，只在套装说明中展示，不得作为可选套装卡片或在后续“基础配件”页重复出现。
- 套装卡片底部使用清晰的勾选状态表示当前唯一选择；方案模式图实时复用该套装图片。
- 第二步的 Extension Bracket 按首步实际所选 C53 套装动态筛选：只展示相同的左右安装侧别与黑／白颜色；例如只选白色左侧 C53 时，仅展示白色左侧支架。切换或取消某一侧套装时，已不再适配的支架必须从清单中移除。
- **Standalone**：C53 独立运行，必须带 GPS；向导仅保留 `3130000100046`（GOTOP）这一料号。后续仍在 C53 自身页面选择前盲区摄像头、屏幕、报警器与服务线材，并在 C53 复核页导出。
- **Cascade**：选择上级主机 AD Plus 2.0、M1N 2.0 或 M3N（三选一并展示主机图）；完成 C53 配置后，直接进入所选主机的原有完整向导，C53 与主机物料在主机复核页合并导出。
- 级联时不选择 C53 GPS，定位由上级 MDVR 提供。
- C53 级联预占上级主机 **1 路 IPC 接口**，不占内置算法；录像通道预占取决于前盲区摄像头（CA51）：未选 CA51 时占 **2 路录像通道**，选 CA51 时占 **3 路录像通道**。预占必须计入主机摄像头页面的剩余额度，禁止超额选择。
